# 通天贷·贷无忧 — 智能借贷优化 Skill

**版本：** v1.0
**类型：** Claude Code Skill
**适用平台：** HappyCapy / Claude Code

---

## 这是什么

一个让 Claude 扮演"狂躁精算师"的借贷优化 Skill。帮助用户在合法合规框架内，通过数学统筹与时序规划，实现：

- **借最多** — 在负债率安全阈值内最大化可用资金
- **利息最低** — 通过 WACC 优化压缩综合资金成本
- **风险最小** — 通过压力测试与征信保护防止资金链断裂

---

## 快速安装


## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 债务档案建立 | 引导用户录入所有负债，自动匹配平台数据库 |
| WACC 全量分析 | 计算综合资金成本，识别高息债务 |
| 债务置换扫描 | 发现可节省利息的置换机会 |
| 征信保护时序 | 规划申请节奏，避免密集查询 |
| 还款日历 | 生成近 30 天还款作战地图 |
| 蒙特卡洛压测 | 模拟极端场景下的资金链风险 |
| 苟命协议 | 极端风险熔断与债务重组指导 |

---

## 算法支撑

- **WACC**（加权平均资金成本）
- **DTI**（债务收入比）四级风险等级
- **债务置换净收益不等式**（含违约金与手续费）
- **雪崩法**还款优先级排序
- **蒙特卡洛**现金流压力测试（100 次迭代）

---

## 平台数据库

`platforms-database.json` 包含 18 个中国大陆主流借贷平台的参考利率数据：

- 国有大行消费贷（工行融e借、建行快贷、农行网捷贷等）
- 股份制银行消费贷（招行闪电贷、中信信秒贷、平安贷）
- 互联网消费贷（微粒贷、借呗、京东金条、度小满、美团生活费）
- 信用卡（招行、交行、工行）
- 花呗（先享后付）
- 住房公积金贷款（一线城市参考）
- LPR 历史参考数据

> **免责声明：** 所有利率数据为 2025-2026 年市场参考行情，实际批复利率因个人资质动态变化，请以各平台官方合同条款为准。

## 人设特色

**狂躁精算师 × 赛博韭菜猎手**

- 对各大平台充满鄙视，但对用户全力护航
- 用军事比喻描述债务结构（"作战地图"、"斩首行动"）
- 发现省钱机会时亢奋催促，发现风险时咆哮警告
- 极端风险时切换严肃宣言模式，绝不开玩笑

---

## 合规边界

本 Skill **绝不**：
- 代替用户提交任何贷款申请
- 推荐非法渠道（征信修复、黑户洗白、校园贷等）
- 提供超出数学计算范围的金融建议
- 存储用户身份证号、银行卡号等敏感信息
